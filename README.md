# German River Gauges

Home Assistant integration for German river water levels.

Supports major rivers in Germany:
- Rhein
- Mosel
- Main
- Donau

## Features
- Water level
- Flow rate
- Temperature (if available)
- Trend
- Flood warning levels
- Config Flow (no YAML)
- 15-minute updates
- HACS compatible

## Installation
Install via HACS as a custom repository.
